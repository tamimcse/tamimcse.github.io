Useful Softwares

My platform: Ubuntu 18.04

  • NetBeans: My favorite IDE (these days I almost exclusively code in C). I found IDE essential for existing projects. However I use gedit and handwritten Makefile when I start a project from scratch.
  • SmartGit: Git Client
  • gnuplot: For creating charts and graphs (for research purposes).
  • TeXstudio: Latex editor (useful for writing research papers and making presentations)
  • LibraOffice: I write all my documents in Latex. But if I need to create images, I found LibraOffice quite good. It's also compatible to Microsoft Office.
