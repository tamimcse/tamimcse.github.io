Useful tools

Platform: Ubuntu 18.04.

I use following tools in my daily life.
  • Gedit: I almost exclusively code in C. If I start a project from scratch, I found gedit (with handwritten make file or commandline compiler) the most convinient
  • NetBeans: My favorite IDE. If I work on an existing C/C++ projects, I found an IDE essential.
  • SmartGit: Git Client
  • gnuplot: For creating charts and graphs (for research purposes).
  • TeXstudio: Latex editor (useful for writing research papers and making presentations)
  • LibraOffice: I write all my documents in Latex. But if I need to create images, I found LibraOffice quite good. It's also compatible to Microsoft Office.
I found following tools/frameworks very useful for networking research:
  • Mininet: Network emulator. It enables us to create a custom network inside a laptop.
  • GStreamer: Video streaming framework.
  • Domino: Router data plane programming in C
I found following tools very useful for high-level synthesis research:
  • Aladdin: ASIC simulator
  • Bambu: High-level synthesis (HLS). It generates Verilog from C code for both FPGA and ASIC
  • OpenROAD: EDA tool. It generates physical chip layout from Verilog
