Open Source Projects

I'm working on Linux kernel's networking subsystem for quite sometime. Besides Linux kernel, I used (sometimes contributed to) several other open source projects. Those include:

  • Mininet: Network emulator. It creates a network inside a laptop.
  • GStreamer: For developing video streaming applications
  • Domino: Dataplane programming using C/C++
  • Aladdin: ASIC simulator
  • ns-3: Network simulator. I found it very useful. But no longer using it. I would rather use Mininet, which is a network emulator
  • P4: Dataplane programming language.

These are some useful tools for my daily life (I'm happily using Ubuntu Gnome 16.04 for past several years. Not going back to Windows/MAC in the foreseeable future).

  • gnuplot: For creating charts and graphs (for research purposes).
  • TeXstudio: Latex editor (useful for writing research papers and making presentations)
  • SmartGit: Git Client
  • NetBeans: My IDE of choice for C/C++ :-) I found IDE essential for larger projects such as Linux kernel. I use gedit and handwritten Make file for smaller projects though
  • Firefox: My favorite browser, although Chrome comes to a close second :-)
  • LibraOffice: I write all my documents in Latex. But if I need to create images, I found LibraOffice quite good. It's also compatible to Microsoft Office
